at line 72
public void parse()

Parse and tranform the preset statement to a ready-to-use information

The statement string can be constructed as (without spaces):

position : info : media : src

where position can be an integer or a string like top or bottom, info can be min for already minified stylesheets, media can be any media string (no validation) and src is the relative path of the file in the original package. Position is a [-1;100] integer range where 100 is the top of the stack (first files to include).

By default, position is 0 (the file is added to the stack), and the script is considered not minified neither as packed.
